Ayuda Gestión de stock

Elementos de el programa

En esta guía de ayuda se verán los diversos métodos y funciones disponibles en el mismo, añadir un producto a la base de datos, buscarlo y modificar dicho recurso y por último listar todo el contenido a traves de un informe, para poder guardarlo en diferentes formatos.

Añadir

En este apartado del programa, a traves de una ventana personalizada, se podrá añadir un nuevo producto,cliente o pedido a la base de datos, en esta ventana se dispondrá de todas las caracteristicas necesarias para añadir el mismo

Busqueda o modificacion

En este apartado del programa, a traves de una ventana personalizada, se podrá buscar y posteriormente modificar cualquier característica del producto,cliente o pedido buscado.

Ver

En este apartado del programa, se generará un informe a través de jasperReports con todos los datos que forman parte de la base de datos con la que se trabaja.
