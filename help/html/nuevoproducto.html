Añadir

Para definir un nuevo producto,cliente o pedido utilizaremos esta ventana, la cual nos proporciona todo lo necesario para realizarlo.

Pasos a seguir

Pulsar sobre el botón de nuevo producto, nuevo cliente o un nuevo pedido en la ventana principal, una vez pulsado aparecerá la ventana de añadir un producto, rellenar los campos con los datos que se consideren y posteriormente pulsar el botón de añadir producto, un cliente o un pedido.
